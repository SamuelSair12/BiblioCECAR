Bienvenido a BiblioCECAR

Plataforma web para la administración integral de la biblioteca de la Corporación Universitaria del Caribe. Gestiona libros, clientes, inventario y ventas desde un solo lugar.

Ver módulos ↓
📚

Módulos del Sistema

Accede a las herramientas de gestión según tu rol asignado.

🏠

Panel Principal

Visualiza indicadores generales, resumen de libros, ventas e inventario en tiempo real.

👥

Usuarios

Registra, edita y activa cuentas de usuario. Controla roles: Administrador, Bibliotecario e Invitado.

📖

Libros

Administra el catálogo de libros: registro, búsqueda por nombre, nivel o edición y consulta de stock.

🧑‍💼

Clientes

Gestiona la base de clientes de la biblioteca: registro, búsqueda y actualización de datos de contacto.

📥

Ingresos

Registra entradas de libros al inventario por lote. Consulta el historial filtrando por fechas o lote.

📦

Inventario

Consulta el stock general o por lote. Identifica de forma visual los libros con existencias bajas o agotadas.

🛒

Ventas

Registra ventas seleccionando cliente y libros. Busca el historial por rango de fechas o cliente.

🏷️

Lotes

Crea y consulta los lotes asociados a libros e ingresos de inventario para una trazabilidad completa.

Acerca del Proyecto

BiblioCECAR es un proyecto académico desarrollado para la asignatura Programación y Tecnologías Web I.

🎯 Objetivo

Desarrollar una aplicación web frontend que consuma una API REST para la gestión integral de la biblioteca de CECAR, aplicando buenas prácticas de desarrollo web.

🛠️ Tecnologías

  • HTML5 semántico
  • CSS3 · Flexbox · Grid
  • JavaScript ES2024 (Vanilla)
  • API REST · Fetch
  • GitHub Pages · Docker
